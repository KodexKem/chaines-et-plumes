# Résumé des principales balises HTML

## Titres : `<h1>` à `<h6>`

- Utilisés pour structurer le contenu en titres et sous-titres.
- `<h1>` est le plus important, `<h6>` le moins important.

## Paragraphe : `<p>`

- Sert à écrire un bloc de texte.

## Image : `<img>`

- Permet d’afficher une image.
- Utilise des attributs comme `src` (source) et `alt` (texte alternatif).
  ⚠️ Il est primordial de paramétrer l'alt! (Aide personnes malvoyantes, Debug, Référencement).

## Saut de ligne : `<br>`

- Insère un retour à la ligne sans créer un nouveau paragraphe.

## Listes

- `<ol>` : crée une liste ordonnée (numérotée).
- `<ul>` : crée une liste non ordonnée (à puces).
- `<li>` : définit un élément d’une liste.
