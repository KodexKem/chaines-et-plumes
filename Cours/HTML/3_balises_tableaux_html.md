# Les balises relatives aux tableaux en HTML

## `<table>`
- Balise principale qui définit un tableau.

## `<tr>`
- Définit une ligne dans le tableau.

## `<th>`
- Définit une cellule d’en-tête (titre de colonne ou de ligne).
- Le texte y est généralement affiché en gras et centré par défaut.

## `<td>`
- Définit une cellule de données (contenu du tableau).

## `<thead>`
- Regroupe les lignes d’en-tête du tableau.
- Placé généralement au début du tableau.

## `<tbody>`
- Regroupe les lignes principales du tableau (le corps).
- Utilisé pour structurer logiquement le contenu.

## `<tfoot>`
- Regroupe les lignes de bas de tableau (souvent pour des totaux ou résumés).
- Placé logiquement après `<tbody>`, mais peut s’afficher en bas même si défini avant.

## Attributs de cellules

### `rowspan`
- Permet à une cellule de s’étendre sur plusieurs lignes.

### `colspan`
- Permet à une cellule de s’étendre sur plusieurs colonnes.
