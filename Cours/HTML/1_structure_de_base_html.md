# Structure de base d’un fichier HTML

## `<!DOCTYPE html>`
- Indique au navigateur que le document est un fichier HTML5.
- Doit être placé au tout début du fichier.

## `<html>`
- Conteneur principal de la page web.
- Englobe tout le contenu du document (sauf `<!DOCTYPE html>`).

## `<head>`
- Partie réservée aux informations sur la page (métadonnées).
- Contient par exemple le titre (`<title>`), l’encodage des caractères, les feuilles de style (CSS) ou les scripts.

## `<body>`
- Partie visible par l’utilisateur.
- Contient le contenu principal de la page : textes, images, liens, tableaux, formulaires, etc.
