# La balise `<link>` en HTML

## Rôle
- Permet de lier un document externe à la page HTML.
- Utilisée principalement pour relier une feuille de style CSS.

## Attributs importants
- `rel` : définit la relation avec le document lié (ex. `stylesheet` pour une feuille CSS).
- `href` : chemin vers le fichier à lier.
- `type` : indique le type du fichier lié (optionnel, souvent `text/css` pour une feuille de style).

# Les classes et ID en CSS

- Les **classes** et les **ID** sont des **attributs en HTML** qui permettent d’identifier ou de regrouper des éléments.

## Les classes
- Une classe permet de regrouper plusieurs éléments pour leur appliquer un même style.
- En CSS, elles se définissent avec un point `.` suivi du nom de la classe.
- Un élément peut avoir plusieurs classes.

## Les ID
- Un ID identifie un élément unique dans une page.
- En CSS, ils se définissent avec un dièse `#` suivi du nom de l’ID.
- Chaque ID doit être unique dans une page.

---

# Les sélecteurs CSS de base

- `*` : sélectionne tous les éléments.
- `element` : sélectionne tous les éléments d’un type donné (ex. `p` pour les paragraphes).
- `.classe` : sélectionne tous les éléments qui ont une classe donnée.
- `#id` : sélectionne l’élément qui a un identifiant donné.
